# The Instagram Photo Feed (with Bootstrap)

This project is a completed implementation of an **Instagram-style photo feed**, built using **HTML, CSS, and Bootstrap**.

The main focus of this project was to practice the construction of layouts using the **Bootstrap grid system**, as well as the integration of common UI components such as modals and dropdown menus, replicating the behavior and structure of a real social media feed.
